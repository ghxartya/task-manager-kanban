# Task Manager Kanban

### The application has been created via Vite using the react-ts template flag and was downgraded to React 18.

## Startup Instructions

1. Install dependencies: `yarn install`
2. (Optional) Run dev server: `yarn dev` (starts at http://localhost:5173)
3. Build for production: `yarn build`
4. Preview build: `yarn preview` (starts at http://localhost:4173)
